# 작업 디렉토리 생성방법

```
./qq 문제번호
./qq 문제링크
./sqq 문제번호
./sqq 문제링크
```

## 예시

```
./qq 1008
./qq https://www.acmicpc.net/problem/1008
./sqq 1008
./sqq https://www.acmicpc.net/problem/1008
```

# 샘플 수정 방법

- `.sample` 디렉토리 내 파일 수정

# STANDARD 문제 구분

- 문제 난이도의 기준이 되는 문제
- 문제제목 앞에 ★ 추가

# sqq

- 디렉토리 이름이 문제명이 아니라 SHA1 값으로 바뀜
